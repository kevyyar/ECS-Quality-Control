# Issue tracker

This repo tracks issues in GitHub Issues (`gh issue`).
Repository: `kevyyar/ecs-qc`

- Create: `gh issue create`
- View/List: `gh issue list`
- Update: `gh issue edit`, `gh issue close`, `gh issue reopen`
- Labels are expected to be present in the repo and are mapped in `docs/agents/triage-labels.md`.
- If no issue labels exist yet, create them on first use as needed.
